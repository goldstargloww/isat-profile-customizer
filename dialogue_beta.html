ISAT Prologue Maker - made by kongkrog(lea) - all assets belong to insertdisc5

Image should be roughly the same size as ISAT's face sprite. [PS] tag is a bit buggy right now.

Tag guide:

[SW]...[/SW]: Applies a wave effect.

[PS=x]: Introduces a pause. The x specifies the duration of the pause.

[SHAKE]...[/SHAKE]: Applies a shake effect.

[IMAGE1], [IMAGE2], [IMAGE3], [IMAGE4]: Change the dialogue image to that specific image associated with tag.

Font in px | Delay in miliseconds | Tags are not combinable.

Default dialogue speed is 25.

This will be the default [IMAGE1]. ISAT Faces directory: In Stars And Time/www/img/faces

Image tags: [IMAGE2], [IMAGE3], [IMAGE4] (All Optional)

Image tags: [IMAGE5], [IMAGE6], [IMAGE7] (All Optional)

Use these tags respectively to change image on the fly.

Transparent rendering will only work on Chromium browsers, not Firefox.

If there are artifacts, re-offseting the dialogue image might work.
