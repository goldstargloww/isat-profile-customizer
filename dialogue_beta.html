In Stars And Time Dialogue Maker

Made by kongkrog(lea)

All assets belong to insertdisc5

Due to how the ISAT dialogue image works, any image larger than 500px will be scaled down to 395px.

Any image's height smaller than 500px will stay unscale. The gif will auto-crop to the highest image height.

Made using gif.js and gifsicle (the browser version).

Tag and Description Image
[SW]...[/SW] | [SHAKE]...[/SHAKE]:
Applies a wave/shake effect.

Example: [SW]Wavey text[/SW]. [SHAKE]Shaky text[/SHAKE]
[B]...[/B] | [I]...[/I]:
Format the text in bold / italic.

Example: [B]Bold text[/B]. [I]Italic text[/I].
[BR]:
Force a line break.

Example: a line[BR]another line[BR]another another line
[PS=x]:
Introduces a pause. The x(ms) specifies the duration of the pause.

Example: 300 [PS=300]500 [PS=500]1000 [PS=1000]2000 [PS=2000]End.
[IMAGE1], ..., [IMAGE13] | [BIMAGE1], ..., [BIMAGE7]:
Change the dialogue | background image to that associated with tag.

Example: OMG BONIFACE YYAYAYAY!!!! [IMAGE2]HMPFFF!
[IMAGExT]:
Add a "T" in the IMAGE tag to trigger the switching portrait animation.

Example: Can I talk using your Fighter? [IMAGE2T]Or the Kid?
[CLEAR] | [SCLEAR] | [BCLEAR]:
Remove all text and change font size to default | small | big.
Effects are compatible to different font size.

Example: normal text[CLEAR]another text[BCLEAR]big text.[SCLEAR]smol text.
[CLRIMG]:
Clear text, remove current image and reset back to normal dialogue box.

Example: i'm gonna be gone[PS=500][CLRIMG]aaa
[DIALBOX=x]:
Clear text, remove current image and add dialogue name. Set dialogue name to x(text), set x to '' to remove it.

Example: i'm gonna turn into another person[PS=500][DIALBOX=person]hi.

Tags are not combinable.

This will be the default [IMAGE1]. ISAT Faces directory: In Stars And Time/www/img/faces

Image tags: [IMAGE2], [IMAGE3], [IMAGE4], [IMAGE5], [IMAGE6], [IMAGE7] (All Optional)

Image tags: [IMAGE8], [IMAGE9], [IMAGE10], [IMAGE11], [IMAGE12], [IMAGE13] (All Optional)

Add a "T" in the tag ([IMAGExT]) (x is a number) to run the portrait switch animation.

Default [BIMAGE1]. Background image's size should be 816x650 or higher (1.26:1) else it'll warp itself to the background.

Background image tags: [BIMAGE2], [BIMAGE3], [BIMAGE4], [BIMAGE5], [BIMAGE6], [BIMAGE7] (All Optional)

Character Name Box will only appear if you clear image ([IMAGE1] is empty).

More images with lower height will fully scale up if "Adjust Image Limit" is low.

If discord is crabbing up gif transparency, enable "Disable Transparency Optimization?" might work.

If transparent rendering doesn't work on Firefox, switching to Chromium browers might work.
