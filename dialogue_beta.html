ISAT Dialogue Maker - made by kongkrog(lea) - all assets belong to insertdisc5

Image should be roughly the same size as ISAT's face sprite.

Made using gif.js and gifsicle (the browser version).

Tag guide (Read this):

[SW]...[/SW]: Applies a wave effect.

[B]...[/B] | [I]...[/I]: Format the text in bold / italic.

[PS=x]: Introduces a pause. The x(ms) specifies the duration of the pause.

[SHAKE]...[/SHAKE]: Applies a shake effect.

[IMAGE1], ..., [IMAGE7]: Change the dialogue image to that specific image associated with tag.

[BIMAGE1], ..., [BIMAGE7]: Change the background image to that specific image associated with tag.

[CLEAR]: Remove all text and reset the dialogue box. You can make multiple dialogue in a same gif this way.

Tags are not combinable.

This will be the default [IMAGE1]. ISAT Faces directory: In Stars And Time/www/img/faces

Image tags: [IMAGE2], [IMAGE3], [IMAGE4], [IMAGE5], [IMAGE6], [IMAGE7] (All Optional)

Default [BIMAGE1]. Background image's size must be 816x650 or higher but same aspect ratio (1.26:1).

Background image tags: [BIMAGE2], [BIMAGE3], [BIMAGE4], [BIMAGE5], [BIMAGE6], [BIMAGE7] (All Optional)

Character Name Box will only appear if you clear image ([IMAGE1] is empty).
